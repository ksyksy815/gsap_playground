Stagger04

Stagger is GSAP's way of automatically spacing animations, making a group of elements animate one after another beautifully - with just a single line of code.

In GSAP, stagger is a special property that spaces out the starting time of animations across multiple elements automatically.

Read more about Staggers

Example 1

gsap.to(".box", { scale: 1, duration: 1, stagger: 0.05, });

Example 2

gsap.from(".box", { s scale: 0, opacity: 0, immediateRender: false, duration: 1, stagger: 0.1, });
